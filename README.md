rnaseq_install - contains instructions and files required for installing nf-core/rnaseq on AWS HealthOmics as a private workflow.  
Used to work with nf-core/rnaseq v3.14.0, but may work with newer versions.  
Download and open rnaseq_install.zip, and follow the instructions in the rnaseq_install.  
User_managment - helper app for adding all necessary permissions for new users.  
Download the User_managamenr.zip open it, and run main.py.
